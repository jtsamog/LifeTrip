# LifeTrip

#### LifeTrip is a prototype I made to test Amadeus API <[Flight Low-Fare Search](https://sandbox.amadeus.com/travel-innovation-sandbox/apis/get/flights/low-fare-search)>.
#### Amadeus provides a sandbox to test different APIs which give data (like airlines , rental cars and hotels ) for tourism agencies  <[Amadeus APIs List](https://sandbox.amadeus.com/getting-started)>. 

#### Please register for Amadeus Api Key at https://sandbox.amadeus.com/getting-started, then update the value of API_KEY in ApiManager
